# meu-portfolio

🌐 Portfólio pessoal desenvolvido com HTML, CSS e JavaScript para apresentar meus projetos, habilidades e trajetória na área de tecnologia.

## 🚀 Sobre o projeto

O portfólio foi criado para servir como uma vitrine profissional, reunindo informações sobre mim, minhas competências técnicas e projetos desenvolvidos.

A proposta do site é oferecer uma navegação simples, moderna e responsiva, proporcionando uma boa experiência tanto em desktop quanto em dispositivos móveis.

## 👨‍💻 Sobre mim

Sou graduado em Análise e Desenvolvimento de Sistemas, com formação técnica em Manutenção e Suporte em Informática.

Tenho afinidade com desenvolvimento em Java, principalmente para aplicações Desktop, e estou em constante evolução, expandindo meus conhecimentos para o desenvolvimento Web.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git e GitHub

- ## 📸 Funcionalidades

- Apresentação pessoal
- Seção "Sobre mim"
- Listagem de habilidades
- Exibição de projetos
- Links para redes sociais
- Layout responsivo
